# 🚀 Pipeline de Limpieza de Ventas (E-commerce)

Este proyecto automatiza la limpieza de datos de ventas mensuales y los carga en una base de datos SQL para su análisis.

## 🛠️ Herramientas utilizadas
* **Python**: Limpieza y transformación con Pandas.
* **SQL**: Almacenamiento y consultas relacionales con SQLite y SQLAlchemy.

## 📈 Resultados
* Se eliminaron duplicados y se corrigieron formatos de fecha.
* Se calculó el ingreso total por transacción uniendo tablas de ventas y productos.
